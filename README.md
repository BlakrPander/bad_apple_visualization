# bad_apple_visualization
Final result is uploaded to BiliBili[https://www.bilibili.com/video/BV1CY49zCE1e/]

# Run code
To run the code and re-process the result, you need package:
- opencv
Pull the github repository and configure env with the Cmakelist.txt.
Generate binary file with Main.cpp.
Download the original "Bad Apple" video from **/resource** and put it in the same directory with the binary Visualization.
Then run the binary.
